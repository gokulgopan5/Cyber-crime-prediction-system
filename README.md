# Cyber-crime-prediction-system
📌 Overview
The Cybercrime Prediction System is a machine learning-based web application designed to detect fraudulent transactions and visualize cybercrime hotspots. The system uses the Random Forest algorithm to classify transactions as fraudulent or legitimate based on transaction details. It also provides an interactive map to identify cybercrime-prone locations, helping organizations take proactive security measures.

✨ Features
🔍 Machine Learning-based fraud detection
📊 Random Forest classification model
📍 Geographic hotspot mapping
⚡ Real-time fraud prediction
🌐 Interactive web interface
☁️ Firebase database integration
📈 Transaction history and analytics
🛡️ Risk score generation for each transaction

⚙️ How It Works
User enters transaction details.
Data is preprocessed and transformed.
The trained Random Forest model predicts whether the transaction is fraudulent.
A risk score is generated.
Fraudulent transactions are stored in Firebase.
Locations are displayed on an interactive hotspot map.
